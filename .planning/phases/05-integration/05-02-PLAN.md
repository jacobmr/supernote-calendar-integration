   - Create 2 recurring meetings (different series) + 2 ad-hoc meetings

   - Verify: 4 mappings total in store
   - Run all 4 through NoteCreator.processNewMeetings
   - Verify: 4 notes uploaded total
   - Verify: recurring notes have date-only filenames, ad-hoc notes have date+time filenames

3. **"should not re-create note for ad-hoc meeting on second run"**
   - Process ad-hoc meeting through full pipeline
   - Process same meeting through NoteCreator again
   - Verify: second run returns { created: 0, skipped: 1 }

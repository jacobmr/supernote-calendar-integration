**How OAuth bootstrap works:**
1. `npm run dev` runs src/index.ts locally, which calls authenticate()
2. authenticate() opens an auth URL and starts a callback server on localhost:3000
3. User visits auth URL in browser, grants consent, Google redirects to localhost:3000/oauth/callback
4. tokens.json is created locally with access_token + refresh_token
5. Copy tokens.json to server's Docker volume at /app/data/tokens.json
6. The app auto-refreshes tokens using the refresh_token — no manual re-auth needed

**TOKEN_STORE_PATH:** Set to /app/data/tokens.json in production .env (from Plan 08-01)

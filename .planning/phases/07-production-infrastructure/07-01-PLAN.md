  <name>Task 1: Install Docker and configure server basics</name>
  <files>N/A — remote server commands via SSH</files>
  <action>
SSH into 37.27.198.218 and perform initial server setup:

1. Update system packages: `apt update && apt upgrade -y`
2. Install Docker Engine via official apt repository (NOT snap):
   - Add Docker GPG key and repository
   - `apt install docker-ce docker-ce-cli containerd.io docker-compose-plugin`
   - Verify: `docker run hello-world`
3. Configure UFW firewall:
   - `ufw allow 22/tcp` (SSH)
   - `ufw allow 80/tcp` (HTTP — for ACME challenges)
   - `ufw allow 443/tcp` (HTTPS)
   - `ufw enable`
4. Set timezone: `timedatectl set-timezone America/New_York`
5. Create app directory: `mkdir -p /opt/supernote`

Do NOT install Docker via snap — it has mount namespace issues with Compose volumes.
</action>

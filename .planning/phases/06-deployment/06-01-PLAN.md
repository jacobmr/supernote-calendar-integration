  <action>
  1. Create `docker-compose.yml`:
     - Service `app` building from current Dockerfile
     - Volume mount `./data:/app/data` for state file persistence across container restarts
     - Volume mount `./.env:/app/.env:ro` for environment configuration
     - Port mapping `3000:3000`
     - `restart: unless-stopped` policy
     - Health check: `curl -f http://localhost:3000/health || exit 1` with 30s interval, 10s timeout, 3 retries

2. Add `GET /health` endpoint to `src/index-server.ts`:
   - Return 200 with JSON `{ status: "ok", uptime: process.uptime() }`
   - Place BEFORE authMiddleware so it's publicly accessible (needed for Docker health check)
   - Keep it simple — no dependency checks needed for a self-hosted single-user app

3. Update Dockerfile:
   - Add `curl` to alpine image (`RUN apk add --no-cache curl`) for health check
   - Add `HEALTHCHECK` instruction matching compose config

Do NOT add docker-compose profiles, networks, or multi-service complexity — this is a single-container app.
</action>
